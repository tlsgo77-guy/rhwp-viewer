## 문제 요약

`cargo test` 실행 시 `re_sample_gen.rs`의 테스트 함수들이 `samples/re-*.hwp` 파일을
자동 생성/덮어쓴다. 자동 생성 파일이 git 추적 대상에 포함되어 있어 테스트 실행마다
불필요한 변경이 발생한다.

현재 `git status`에 수정(M)으로 표시된 파일: **90개** (`samples/re-*.hwp`)

## 원인

`re_sample_gen.rs`의 출력 경로가 `samples/re-*.hwp`로 하드코딩되어 있음.
`samples/` 폴더는 git 추적 대상이므로 테스트 실행마다 변경사항이 발생.

## 개선 방향

- `re_sample_gen.rs` 출력 경로를 `samples/re-*.hwp` → `output/re-*.hwp`로 변경
- `samples/re-*.hwp` (현재 수정 표시된 90개) git에서 제거 및 파일 삭제
- `-hancom.hwp` 패턴 파일은 의도적으로 추가된 것이므로 유지
- `output/`은 `.gitignore`에 이미 등록되어 있으므로 추가 작업 불필요

## 구현 범위

- `src/renderer/composer/re_sample_gen.rs` — 출력 경로 변경
- `samples/re-*.hwp` 90개 — `git rm` 및 파일 삭제

## 구현 단계

| 단계 | 내용 |
|------|------|

## 기대 효과

- `cargo test` 실행 후 git 변경사항 없음
- 자동 생성 파일은 `output/`에 생성되어 git 추적 제외
