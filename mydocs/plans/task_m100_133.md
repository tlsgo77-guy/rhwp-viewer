### 0단계: devel → main PR 생성 및 merge

Task #132까지 완료된 `local/devel`을 `devel`에 merge하고,
`devel → main` PR을 생성·merge한다.

- `local/devel`에 local/task132 merge (Task #132 포함)
- `devel` push
- `devel → main` PR 생성 → 승인 → merge

- GitHub Actions `release.yml` (있는 경우) 확인
